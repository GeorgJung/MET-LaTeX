Usage

  • Install a fairly complete version of TeX/LaTeX, able to run pdflatex.
  • Copy newcsen.cls (and optionally GUC-logo.pdf) to a place where LaTeX can find it.
  • Set up a config.tex file for the class you teach. It should be stored in a central place where LaTeX can find it or inside a directory that contains the other LaTeX sources for a specific course. The individual LaTeX-files for your assessments can be in subdirectories of the one containing config.tex (up to two levels).
  • Write your assessments, with newcsen as document style and the respective options (see below).

Documantation

Files

  • newcsen.cls - New class file to compile course documents.
  • config.tex - Example configuration file to test the class.
  • PracticeAssignment01.tex - Example assessment file to test the class.

Document class options:

General

  • solution - show solution

Type

  • practice - compile a practice assignment
  • assignment - compile an assignment
  • popquiz - compile a popquiz
  • quiz - compile a quiz
  • mock - compile a mock exam
  • midterm - compile the midterm exam
  • final - compile the final exam
  • makeup - compile the makeup exam

Logo?

  • logo - add a logo to the output

Formula sheets

  • formula - add a formula sheet at the end of the assessment (necessary if formulas are referenced!)

Details

  • nomarktbl - suppress marking table on first page (for marked assessments)
  • noid - suppress student identification form (e.g., for computerized assessments)
  • nobarcode - suppress barcode placeholder (e.g., for computerized assessments)
  • idform - print student id-form on cover page instead of barcode
  • nonumber - suppress numbering
  • coverpage - generate a cover page (default for quiz, mock exam, midterm, final, and makeup)
  • nocover - do not generate a cover page (default for practice assignment, assignment, popquiz)
  • noinstruct - suppress instructions on cover page

Commands for config.tex

  • \Institute{} - sets the institute (defaults to "The German University in Cairo")
  • \Department{} - sets the department (defaults to "Dpt. for Media Engineering and Technology")
  • \Class{} - sets the class code
  • \Title{} - sets the title of the class
  • \Edition{} - sets the semester for this class
  • \Lecturer[]{} - sets the class lecturer (optional short name)
  • \Email{} - sets the e-mail address of the lecturer
  • \Logo{} - sets the base name of an image file that contains a logo (defaults to "GUC-logo")

Commands for individual files

  • \Assessmenttype{} - sets an individual assessment type if necessary (e.g., "Practical Assignment", usually set through appropriate option)
  • \Number{} - sets the number of this assessment (usually set automatically from filename, e.g., use "PracticeAssigment02.c.tex" for practice assignment 2, version 3)
  • \Topic{} - sets the topic of the exercise set
  • \Version{} - sets the version number, necessary if multiple versions are prepared (not set automatically!)
  • \Date{} - sets the discussion or submission or exam date
  • \Pages - returns the total number of pages
  • \Scratchpaper{} - sets the number of pages for jotting paper in the end (defaults to three for marked exam types, zero otherwise)
  • \Scratchpages - returns the number of pages for jotting paper in the end
  • \Duration{} - sets the duration of the exam (defaults to 120 min for midterm and 180 min for final and makeup)
  • \Instructions{} - sets instructions other than the default for this exam
  • \Management{} - adds content of braces to front/cover-page to include extra administrative information (_e.g._, for distinguishing majors or type of the exam such as final, mid-final, or second makeup)
  • \AddFormula[]{} - adds a formula for the formula sheet (optional title, should be set with the current version due to a bug)
  • \SolutionSpace{} - sets a space to be left for the solution if the solution is not shown
  • \Mark{} - set marks for sub-exercises (checks if sum is correct, but no automatic adding)
  • \Bonus{} - same as \Mark{}, but sets bold font mark which is not counted (bonus mark)
  • \checkedItem - used instead of \item for correct answers in mcq-environment (check-mark appears when solution option is set

Commands for unofficial use only

  • \Exercisename{} - Sets a name for the exercise header (default: Exercise)
  • \Bonusname{} - Sets a name for the bonus exercise header (default: Bonus Exercise)
  • \Exerciseheadskip{} - Sets a distance between number, theme and title
  • \Exerciseskip{} - Sets the distance between two exercises

Environments

\begin{exercise}{ marks }{ theme }{ title } ... \end{exercise}

Standard exercise environment.

\begin{bonusexercise}{ marks }{ theme }{ title } ... \end{bonusexercise}

Environment for bonus exercises (not counted into the marking table)

\begin{solution} ... \end{solution}

Solutions

\begin{unsolved} ... \end{unsolved}

Material that should disappear when solutions are given

\begin{mcq} ... \end{mcq}

Environment for multiple-choice questions. Use \item for the individual answers, use \checkedItem for individual answers that should appear check-marked in the solution.

Notes

I am experienced with using LaTeX, but I am not experienced with style authoring. Some commands should be environments and vice versa, various improvements can be made. However, the style has proven very useful to me. Use (and improve) as you see fit.
